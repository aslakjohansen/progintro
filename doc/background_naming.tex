\section{Naming}

% intro: importance, not about coming up with good names for stuff, about the ability to reference stuff and how the mechanics of that ability work, in this book we will cover several such mechanisms
Naming is a concept that is central to computer science. In this context, it has nothing to do with coming up with good names for things. Instead, it refers to the ability to reference things. In programming, we rely on a number of mechanisms for looking up references to things. In this book, we cover several of these.
