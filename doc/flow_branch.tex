\subsection{Blocks}

% problem: technically branching on a single statement is all we need, but it is not particularly convenient, imagine us having to do two things if a condition holds true: we will have to check a condition and if true do the first thing, then we will have to check the same condition again and if true do the second thing, this comes with three significant downsides, namely: (i) the condition needs to be evaluated twice and this takes time and could involve complex side effects, (ii) the code would quickly get complex to understand, and (iii) it would get significantly more convoluted if the first thing modifies a variable that takes part of the condition
