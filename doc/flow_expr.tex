\section{Choices in Expressions}

The perhaps simplest expression of a choice in \csharp\ is that of a \idx{conditional}{Conditional!Expression} expression. In practice, this is implemented by the \idx{ternary operator}{Operator!Ternary}. The syntax definition for that ternary operator can be found in syntax \ref{syntax:flow:expr}. Here, one condition decides which of two expressions to evaluate. These two expressions must be of types that \idx{satisfy}{Type!Satisfiability} the type of the full ternary expression.
