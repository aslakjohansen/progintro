Write a program in which:

\begin{enumerate}
  \item the result of a roll of a dice is stored in a variable named \varname{dice}.
    \begin{itemize}
      \item Which type should the variable be declared as?
      \item Chose a value and initialize the variable to it.
    \end{itemize}
  \item Declare a boolean variable and assign it a value that represent whether the value in \varname{dice} is even and greater than 3.
  \item Print out the value of this boolean variable.
\end{enumerate}
