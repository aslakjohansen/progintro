The following happens:
\begin{itemize}
  \item \varname{i} is printed normally.
  \item \varname{l} is padded to a width of four and printed.
  \item \varname{f} is printed normally.
  \item \varname{d} is padded to a width of six with two decimals and printed.p
\end{itemize}

This is useful for making sure that commas are aligned across multiple lines.
