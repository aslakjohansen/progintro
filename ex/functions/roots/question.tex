Write a program in which:
\begin{enumerate}
  \item There is a function called \funcname{discriminant} that calculates a discriminant\footnote{\url{https://en.wikipedia.org/wiki/Discriminant}} basend on the parameters $a$, $b$ and $c$.
  \item There is a function called \funcname{roots} that takes the paramters $a$, $b$ and $c$ from a quadratic polynomial\footnote{\url{https://en.wikipedia.org/wiki/Quadratic_function}}, and returns an array of roots.
    \begin{itemize}
      \item \textbf{Hint:} You can use \texttt{Math.Sqrt(9.0)} to calculate $\sqrt{9}$.
    \end{itemize}
  \item There is code that demonstrates how to call \funcname{roots} and prints out the result.
\end{enumerate}
